
# 删除有序数组中的重复项：

## 解题思路
1、首先判断数组是否为0
2、定义慢指针和快指针
3、慢指针初始值为0 ，快指针初始值为1
4、开始一轮循环，以快指针为下标遍历数组
5、当快指针下标对应的值与慢指针对应的值不相等时，则用快指针下标对应的值覆盖慢指针下标+1所对应的值
6、循环结束返回慢指针+1，则就是新数组的长度

## JS解题代码：
```
var removeDuplicates = function(nums) {
  if(nums.length == 0){
      return 0
  }else{
      var i = 0;
      for(var j=1;j<nums.length;j++){
          if(nums[i] != nums[j]){
              nums[++i] = nums[j]
          }
      }
      return i+1
  }
};

```

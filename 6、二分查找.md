# 题目设定

## JS题解代码

```
let left = 0;
    let right = nums.length - 1
    while(left <= right){
        let mid = Math.floor(left + (right - left)/2)
        if(nums[mid] === target ){
            return mid
        } else if(nums[mid]< target ){
            left = mid+1
        } else {
            right = mid - 1
        }
    }
    return -1;
```
